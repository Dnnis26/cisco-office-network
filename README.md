# Projeto de Rede Corporativa - Cisco Packet Tracer
Projeto realizado no Cisco Packet Tracer simulando uma rede corporativa com segmentação por VLAN
Este projeto foi desenvolvido utilizando o Cisco Packet Tracer com o objetivo de simular uma rede corporativa, aplicando conceitos de redes de computadores e boas práticas de configuração de equipamentos Cisco.

# Objetivos
Projetar uma topologia de rede funcional.
Configurar o endereçamento IPv4.
Implementar VLANs para segmentação da rede.
Configurar o roteamento entre VLANs.
Garantir a comunicação entre todos os dispositivos.
Realizar testes de conectividade utilizando ping.
